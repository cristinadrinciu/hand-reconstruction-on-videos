# Demo videos

Rendered outputs of the hybrid pipeline on the seven qualitative test
scenarios from Chapter 5 of the thesis. Each clip shows the input footage
with the reconstructed MANO hand meshes overlaid (one colour per tracked
hand), produced by the full WiLoR + HaPTIC pipeline.

| File | Scenario | What it shows |
|------|----------|---------------|
| `01_simple_hands.mp4`   | Simple hands       | baseline gestures, clear frontal hands |
| `02_occlusion.mp4`      | Occlusion          | a hand gripping objects in a cooking clip |
| `03_fast_motion.mp4`    | Fast motion        | juggling, large frame-to-frame movement |
| `04_multiple_hands.mp4` | Multiple hands     | several people at a table (UNO), up to six hands per frame |
| `05_distance_scale.mp4` | Distance and scale | a hand moving toward and away from the camera |
| `06_orientation.mp4`    | Orientation        | sign-language fingerspelling, varied hand poses |
| `07_long_sequence.mp4`  | Long sequence      | ~45 s continuous guitar clip, temporal stability |

The underlying clips are short excerpts of publicly available videos, used
here only to demonstrate the reconstruction qualitatively for academic
purposes. All rights to the original footage belong to their respective
owners.
